# Saniya Goyal — Software Engineer Portfolio

A minimalist, highly interactive engineering portfolio built with a focus on typography, subtle motion, and a custom pixel-art aesthetic. Designed to feel like an "engineering notebook" that balances clean, professional structure with unique visual character.

## 🚀 Live Demo

[saniya-portfolio.vercel.app](https://saniya-portfolio.vercel.app) *(assuming default Vercel routing)*

## ✨ Features

- **Custom Pixel-Art SVGs**: Hand-crafted SVG icons for achievements and skills, drawn pixel-by-pixel using `shape-rendering="crispEdges"`.
- **Fluid Typography & Layout**: Built entirely with modern Vanilla CSS, utilizing `clamp()`, CSS Grid, and Flexbox for a perfectly responsive experience from mobile to ultrawide displays.
- **Organic Motion**: 
  - **Scroll Stagger**: Elements cascade into view naturally using the `IntersectionObserver` API.
  - **Magnetic Interactions**: Contact buttons and resume links feature a subtle magnetic pull that tracks the user's cursor.
  - **Ambient Glows**: Project cards feature a dynamic radial-gradient glow that perfectly tracks mouse movement.
- **Zero Dependencies**: 100% Vanilla HTML, CSS, and JavaScript. No bulky frameworks, resulting in lightning-fast load times.

## 🛠️ Technologies

- **HTML5**: Semantic document structure.
- **Vanilla CSS3**: Advanced CSS variables (`--accent`, `--bg-elevated`, etc.), Flexbox, CSS Grid, and custom `color-mix()` tinting.
- **Vanilla JavaScript**: Lightweight DOM manipulation, custom cursor logic, and Intersection Observers for performant animations.
- **Lucide Icons**: Clean, consistent vector iconography mixed with custom pixel art.

## 💻 Running Locally

Since this is a static site with no build step or dependencies, running it locally is incredibly simple:

1. Clone the repository:
   ```bash
   git clone https://github.com/san7iya/saniya-portfolio.git
   ```
2. Open the directory:
   ```bash
   cd saniya-portfolio
   ```
3. Open `index.html` in any modern web browser.
